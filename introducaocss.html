Introdução ao CSS

Aprendi sobre as configurações básicas, algumas formas básicas de aplicar o CSS como inline (nas TAGS), internal (no head) e external (arquivo externo) com o uso da TAG LINK CSS. No exemplo deste tópico, será feita uma configuração external (uso da TAG link) com algumas configurações básicas no body, h1, p, etc. Que constarão em todas as páginas desse projeto.

Também vale ressaltar a prioridade em trabalhar com CSS External, uma vez que com muitas páginas, gastariamos muito tempo com configurações e futuramente com mudanças das mesmas, assim sendo configurações para todas as páginas. Mas que se houver uma configuração mais especifica na página especifica, podemos usar o INTERNAL e se for dentro de um paragrafo, por exemplo, usar o INLINE e sempre que esses forem utilizados, os mesmos terão prioridade da configuração.
