Ensinamentos do módulo 01 de HTML5 e CSS

Alguns conhecimentos do curso já se encontram nesta página, como por exemplo as ligações com links internos e outros serão adicionadas dentro das respectivas páginas, como por exemplo: multimidia. Outras páginas constaram apenas para efeito de link. Conceito da TAG link com FAVICON e CSS (external)

Foram aplicados os conceitos de hyperlinks internos com a tag ancora, criação de páginas internas, atributos TARGET _SELF para abrir os links na mesma página e o atributo REL NEXT para indicar que o link será para a próxima página. Nas páginas dos links, foi aplicado o conceito e REL PREV para voltar ao index, paragrafos, títulos, etc.

Também Foram aplicados os conceitos de listas não ordenadas para organizar cada link e o parametro TYPE, para escolha do tipo do destaque diferente do padrão (disc).
