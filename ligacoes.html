Links

Estudei a criação de links com a TAG A (ancora) e o uso dos atributos TARGET (_blanc (abre em nova janela), _self (abre na mesma janela), _top e _parent) e REL (next (próxima página), prev (página anterior), author (indica link p site do autor do texto), external (link para outro site), nofollow(indoca links para sites patrocinados, não endossador pela página)).

Links internos

O usuário é levado para uma página dentro da página, aprendemos que se o link for interno, é interessante a página abrir na mesma janela e para isso usamos TARGET _SELF que é o padrão e para o atributo REL, usamos o next, para uma próxima página e prev, para voltar a anterior, atributos usados nesse projeto.

Links Externos

Aprendemos que para links externos, devemos usar a TARGET _BLANC e REL External, author ou nofollow), caso queiramos que a página seja aberta em uma nova página e para indicar o tipo de página (rel). Muito usado para encaminhar o usuário para outro site, sem fazer ele sair do seu.

Links para download

Para o download, precisamos adicionar o caminho do arquivo no parametro HREF="", adicionar o parametro download="" com o nome do arquivo e o TYPE

href="link na pasta" download="arquivo.zip" type="application/zip"

    Seguem abaixo os TYPEs mais usados:

  • application/zip
  • text/html
  • text/css
  • text/j.scripp
  • video/mp4
  • image.jpeg

    • Obs.: O Site IANA.ORG serve de pesquisa para os TYPEs
